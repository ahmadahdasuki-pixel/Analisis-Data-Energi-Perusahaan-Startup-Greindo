# Analisis & Visualisasi Data Energi Terbarukan Indonesia - Studi Kasus Greindo (SDGs 7: Affordable & Clean Energy)

## Description

Repository ini berisi hasil Final Project yang merupakan bagian dari Project Based Learning (PBL) dengan tema SDGs nomor 7 mengenai energi bersih dan terjangkau. Proyek ini menggunakan dataset Greindo_data.csv yang berisi data operasional energi terbarukan di Indonesia periode 2021-2024, mencakup 5.150 baris data dengan variabel seperti kapasitas terpasang (MW), produksi energi (GWh), populasi terlayani, biaya operasional, emisi karbon yang terpangkas, laju efisiensi, skor investasi, jenis energi, tipe provider, status operasional, dan target pencapaian.

Proses analisis dilakukan secara bertahap mulai dari Business Understanding untuk memahami konteks bisnis perusahaan fiktif Greindo sebagai startup analitik energi, Data Understanding melalui eksplorasi data (EDA) menggunakan berbagai visualisasi seperti bar chart, pie chart, histogram, box plot, dan heatmap, hingga Data Preparation yang mencakup penanganan inconsistent values, missing values, duplicated values, dan outliers menggunakan Python (Pandas, Seaborn, Matplotlib, NumPy) di Google Colaboratory. Setelah data dibersihkan, dilakukan Construct Data dan Data Reduction untuk menghasilkan dataset final yang siap dianalisis, yaitu sebanyak 5.105 baris dan 13 kolom.

Hasil akhir dari proyek ini disajikan dalam bentuk visualisasi dan dashboard interaktif menggunakan Tableau yang menjawab tujuh pertanyaan bisnis utama, antara lain distribusi kapasitas terpasang antar wilayah, kontribusi jenis energi terhadap produksi nasional, tren produksi energi bulanan, hubungan antara kapasitas dan realisasi produksi, persentase status operasional fasilitas, perbandingan pencapaian target berdasarkan tipe provider, serta efisiensi biaya operasional antar wilayah. Repository ini mencakup notebook Jupyter (.ipynb) yang berisi seluruh kode analisis Python, dokumen PBL/brief project, serta laporan final lengkap dalam format PDF yang berisi metodologi, hasil analisis, insight, dan rekomendasi strategis untuk mendukung pengambilan keputusan dalam pengembangan energi terbarukan di Indonesia.
